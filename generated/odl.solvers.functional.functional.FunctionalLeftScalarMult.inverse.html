FunctionalLeftScalarMult.inverse¶

FunctionalLeftScalarMult.inverse¶

Inverse of this operator.

The inverse of scalar * op is given by op.inverse * 1/scalar if scalar != 0. If scalar == 0, the inverse is not defined.

OperatorLeftScalarMult(op, s).inverse == OperatorRightScalarMult(op.inverse, 1/s)

Examples

>>> space = odl.rn(3)
>>> operator = odl.IdentityOperator(space)
>>> left_mul_op = OperatorLeftScalarMult(operator, 3)
>>> left_mul_op.inverse([3, 3, 3])
rn(3).element([1.0, 1.0, 1.0])
