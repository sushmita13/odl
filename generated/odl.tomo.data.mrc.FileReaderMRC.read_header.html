FileReaderMRC.read_header¶

FileReaderMRC.read_header()¶

Read the header from file.

The header is also stored in the header attribute.

Returns:

header : OrderedDict

Header from file, stored in an ordered dictionary, where each entry has the following form:

'name': {'value': value_as_array,
         'offset': offset_in_bytes
         'description': description_string}

All 'value'‘s are numpy.ndarray‘s with at least one dimension. If a 'shape' is given in header_fields, the resulting array is reshaped accordingly.
